## 摘要
多头注意力中的稠密输出投影随模型维度呈平方级增长，显著增加了参数量、内存占用和推理成本。我们建议使用固定的、无参数的 Walsh Hadamard 变换，后接轻量级的可学习仿射重缩放（affine rescaling）来替换该投影。这在每个块中消除了约 25% 的注意力参数，同时通过正交且保范（norm-preserving）的变换保留了全局跨头交互。在不同的模型规模下，我们证明了这种结构化替换在标准基准测试中保持了相当或略优的下游任务性能，同时实现了高达 7% 的总参数减少、8.9% 的峰值内存节省以及在大规模下 6.6% 的吞吐量提升，且效率增益随模型规模、批次大小和序列长度单调增长。有趣的是，我们观察到，与稠密模型相比，基于结构化 Hadamard 的模型在验证损失曲线相对于训练 FLOPs 表现得更陡峭，这表明其在训练过程中具有更优的计算利用率。

## Abstract
The dense output projection in multi-head attention scales quadratically with model dimension, contributing significantly to parameter count, memory footprint, and inference cost. We propose replacing this projection with a fixed, parameter-free Walsh Hadamard Transform followed by a lightweight learnable affine rescaling, eliminating approximately 25 percent of attention parameters per block while preserving global cross head interaction through an orthogonal, norm-preserving transformation. Across different model sizes, we demonstrate that this structured substitution maintains comparable or slightly superior downstream task performance on standard benchmarks, while achieving up to 7 percent aggregate parameter reduction, 8.9 percent peak memory savings, and 6.6 percent throughput improvement at scale, with efficiency gains growing monotonically with model size, batch size, and sequence length. Interestingly, we observe that structured Hadamard-based models exhibit a steeper validation loss curve relative to training FLOPs compared to their dense counterparts, suggesting more favorable compute utilization during training.

### 1. 核心问题与研究动机
*   **核心问题**：在标准 Transformer 中，多头注意力的输出投影层（Output Projection）是一个稠密线性层，其参数量和计算量随模型维度 $d_{model}$ 的平方增长（$O(d^2)$）。
*   **研究动机**：
    *   随着模型规模扩大，该层贡献了大量的参数冗余和计算开销。
    *   现有研究多关注 KV 缓存压缩或 MLP 块优化，而忽略了“头混合”（Head-mixing）过程中的效率问题。
    *   作者挑战了“必须使用全稠密矩阵才能有效结合各头信息”的假设，寻求一种既能保持跨头交互又能显著降低成本的结构化方法。

### 2. 方法论
*   **核心思想**：将可学习的稠密投影矩阵替换为一个固定的、无参数的**沃尔什-哈达玛变换（Walsh-Hadamard Transform, WHT）**，并辅以轻量级的可学习仿射变换。
*   **关键技术细节**：
    *   **结构替换**：标准 MHA 的输出为 $Y = \text{Concat}(\text{head}_1, \dots, \text{head}_h) W_O$。论文将其改为：$\text{Output} = \alpha \odot (Y H) + \beta$。
    *   **Hadamard 矩阵 ($H$)**：一种正交、保范的固定矩阵，元素仅为 $+1$ 或 $-1$。它通过“蝴蝶结构”实现全局信息混合。
    *   **仿射重缩放**：引入两个维度为 $d_{model}$ 的可学习向量 $\alpha$（缩放）和 $\beta$（偏置），以补偿固定变换带来的表达能力损失。
    *   **复杂度降低**：计算复杂度从 $O(d^2)$ 降至 $O(d \log d)$；参数量从 $d^2$ 降至 $2d$。

*   **推理加速**：在 XXL 规模下，解码吞吐量提升了 6.6%，且加速效果随模型变大而增强。

### 7. 优点与亮点
*   **归纳偏置（Inductive Bias）**：Hadamard 变换的正交性强制各头学习互补而非冗余的表示，起到了一种隐式正则化的作用。
*   **硬件友好性**：与直接减小维度（会导致张量形状不对齐）不同，该方法保持了原始张量形状，确保了 Tensor Core 的高效利用。
*   **即插即用**：无需引入复杂的超参数或改变 Transformer 的核心逻辑，易于集成。

### 8. 不足与局限
*   **软件实现瓶颈**：目前的 Fast Hadamard Transform 实现（FWHT）尚未像高度优化的 GEMM（通用矩阵乘法）内核那样经过深度优化，导致实际训练时间的缩短可能不如理论 FLOPs 减少得那么显著。

*   **任务局限**：实验主要集中在语言模型和常识推理，尚未在长文本建模、多模态或代码生成等更复杂的任务上验证其鲁棒性。
