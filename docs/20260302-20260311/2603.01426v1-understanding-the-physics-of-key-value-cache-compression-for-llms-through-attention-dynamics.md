## 摘要
随着大语言模型（LLM）的上下文窗口扩展到 100K+ token，键值（KV）缓存已成为主要的内存瓶颈，近期的方法声称可节省 80-90% 的内存且基准测试性能下降极小。我们认为这些评估忽略了一个结构性问题：注意力不仅是存储，更是路由，保留 KV 对并不保证语义的可访问性。我们提出了一种受物理学启发的观点，将 KV 压缩视为对 token 级路由的受控扰动，并区分了保留、可访问性和利用率。通过探测多实体追踪、消歧、指代消解和多步推理的合成任务，我们发现：适度压缩会降低内部表示质量但准确率损失较小，揭示了冗余性；所有模型在 90% 压缩率附近都表现出剧烈的幻觉安全悬崖，这与全局剔除率（GER）的激增相关，暗示了语义可达性的相变；不同架构的路由动力学存在差异，LLaMA 表现出早期共识和后期多样化，而 Qwen 表现出漏斗状的后期收敛，从而导致了不同的韧性特征。除了擦除之外，我们还识别出了表示刚性，即尽管 token 依然存在，但过度的多头级共识会导致路由灵活性崩溃。这些结果表明，稀疏的 token 路由结构决定了压缩容忍度，将 KV 压缩重新定义为注意力几何的结构性探测，并将长上下文可扩展性与自注意力中的稀疏性和彩票假设联系起来。

## Abstract
As context windows in LLMs scale to 100K+ tokens, the key-value (KV) cache becomes the dominant memory bottleneck, with recent methods claiming 80-90% savings and minimal benchmark degradation. We argue these evaluations miss a structural issue: attention is not just storage but routing, and retaining KV pairs does not guarantee semantic accessibility. We propose a physics-inspired view of KV compression as a controlled perturbation of token-level routing, distinguishing retention, accessibility, and utilization. Using synthetic tasks probing multi-entity tracking, disambiguation, coreference, and multi-hop reasoning, we find that moderate compression degrades internal representations with little accuracy loss, revealing redundancy; all models exhibit a sharp hallucination safety cliff near 90% compression, correlated with spikes in Global Eviction Ratio (GER), suggesting a phase transition in semantic reachability; and architectures differ in routing dynamics, with LLaMA showing early consensus and late diversification, and Qwen showing funnel-like late convergence, leading to distinct resilience profiles. Beyond erasure, we identify representational rigidity, where excessive head-level consensus collapses routing flexibility despite token survival. These results suggest sparse token-route structures govern compression tolerance, reframing KV compression as a structural probe of attention geometry and linking long-context scalability to sparsity and the lottery ticket hypothesis in self-attention.

### 1. 核心问题与研究动机
*   **核心问题**：随着 LLM 上下文窗口扩展，KV 缓存成为内存瓶颈。虽然现有压缩方法声称能节省 80%-90% 内存且准确率下降极小，但作者质疑：**注意力机制不仅是存储，更是路由（Routing）**。仅仅保留 Token 并不意味着它在生成过程中语义可达。
*   **研究动机**：现有的长文本基准测试（如 LongBench）仅报告表面准确率，无法揭示压缩如何破坏内部表示。本研究旨在通过“物理学”视角，探索压缩对 Token 路由、语义可达性和表示稳定性的深层影响。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：将 KV 压缩视为对注意力路由的受控扰动，并提出 **Token-Route Lottery Ticket (TR-LT)** 假设，即在稠密注意力层中存在极少数关键的跨层路由路径，只要这些路径存活，推理就能保持。
*   **关键指标**：
    *   **全局剔除率 (Global Eviction Ratio, GER)**：量化答案关键 Token 是否在所有注意力头中都被删除。如果 GER 激增，意味着路由被彻底切断。
    *   **层级头共识 (Head-level Consensus)**：测量不同注意力头是否关注同一个 Token。低共识意味着多样性，高共识可能导致“表示刚性”。
*   **失败模式分类**：
    *   **表示擦除 (Representational Erasure)**：关键 Token 被全局剔除。
    *   **表示刚性 (Representational Rigidity)**：Token 虽然存在，但由于注意力头过度趋同，导致路由缺乏灵活性，无法有效利用信息。

### 3. 实验设计
*   **数据集/场景**：设计了一套受控的合成数据集，包括：
    *   **基础任务**：实体-属性绑定。
    *   **知识操纵**：测试事实结构的系统性剔除。
    *   **多实体/多存在**：测试位置鲁棒性和跨头路由。
    *   **长上下文/多跳推理 (Hops)**：测试长距离语义链的完整性。
    *   **指代消解 (Coreference)**：检测细粒度的表示一致性。
*   **对比方法与设置**：
    *   **模型**：LLaMA-3/3.2 (3B, 8B) 和 Qwen-2.5 (3B, 7B, 14B)。
    *   **压缩策略**：FINCH (分块剔除) 和 AdaKV (全局自适应剔除)。
    *   **推理模式**：问题无关 (Question-agnostic) vs. 问题感知 (Question-aware)。
*   **线性探测 (Linear Probing)**：在冻结的模型隐藏状态上训练分类器，检测压缩后语义信息是否依然“存在”于残差流中。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了 5 个不同规模的模型（3B 到 14B），2 种主流架构（LLaMA, Qwen），2 种压缩算法，以及从 10% 到 90% 的压缩梯度。
*   **充分性**：实验设计非常充分。除了准确率，还包含了注意力热图分析、层级共识曲线、线性探测 F1 值以及 GER 相关性分析。
*   **客观性**：通过合成数据集排除了自然语言中的干扰因素，能够精准控制变量（如 Token 距离、重复次数），实验结果具有高度的可解释性和客观性。

### 6. 主要结论与发现
*   **安全悬崖 (Safety Cliff)**：在 **90% 压缩率**附近，所有模型都会出现性能骤降。这与 GER 的激增高度相关，标志着语义可达性的“相变”。
*   **架构差异**：
    *   **LLaMA**：表现为“倒漏斗”型，早期层达成共识，后期层多样化。
    *   **Qwen**：表现为“漏斗”型，早期探索，后期层高度收敛。
*   **冗余边际**：中度压缩（<50%）通常不会显著损害准确率，甚至可能通过去除干扰路由而略微提升性能。
*   **语义层级**：不同实体的鲁棒性不同。人名、地点较稳定，而“事件 (Event)”和“生物 (Creature)”等复杂关系表示在压缩下极易崩溃。

### 7. 优点
*   **理论深度**：将 KV 压缩从工程优化问题提升到了“注意力几何”的物理机制高度。
*   **新颖指标**：提出的 GER 指标为评估压缩算法提供了一个比准确率更本质的结构化工具。
*   **机制洞察**：成功区分了“信息存在”与“信息可达”的区别，解释了为什么有时模型内部还有信息却无法生成正确答案。

### 8. 不足与局限
*   **数据集局限**：主要依赖合成数据集。虽然有利于机制分析，但在处理自然语言的模糊性、修辞和复杂语境时，结论的迁移性有待进一步验证。
*   **模型规模**：实验最高只覆盖到 14B 模型，对于 70B 或更大型的模型（其冗余度可能更高），“安全悬崖”的位置可能会发生偏移。
*   **应用限制**：研究侧重于分析而非提出新的压缩算法，虽然为算法设计指明了方向（如需保持路由多样性），但未直接提供一个超越现有 AdaKV 的新方案。
