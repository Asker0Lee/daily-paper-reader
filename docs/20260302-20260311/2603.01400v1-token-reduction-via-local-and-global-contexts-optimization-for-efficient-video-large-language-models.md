## 摘要
视频大语言模型 (VLLMs) 展现出强大的视频理解能力，但由于视觉 Token 的冗余，面临效率低下的问题。现有的剪枝方法主要针对帧内空间冗余，或在 LLM 内部通过浅层开销进行剪枝，导致时空缩减效果不佳，且未能充分利用长上下文的可压缩性。这些方法通常会丢弃合并或剪枝 Token 中细微但具有信息量的上下文。在本文中，我们提出了一种新视角，通过局部-全局最优传输 (AOT) 精细化构建帧内和帧间的 Token 锚点 (Anchors)，从而全面聚合信息丰富的上下文。具体而言，我们首先在注意力机制的引导下，在每帧内建立局部和全局感知的 Token 锚点，随后通过最优传输聚合来自被剪枝 Token 的信息上下文，构建帧内 Token 锚点。接着，基于时间视频片段，将每个片段的第一帧视为关键帧锚点，通过最优传输整合来自连续帧的相似信息，同时保留独特的 Token 以表征时间动态，从而以无需训练的方式实现高效的 Token 缩减。广泛的评估表明，我们提出的 AOT 在主流视频 LLM 的各种短视频和长视频基准测试中均获得了具有竞争力的性能，在保持时间与视觉保真度的同时，显著提升了计算效率。项目主页：https://tyroneli.github.io/AOT。

## Abstract
Video Large Language Models (VLLMs) demonstrate strong video understanding but suffer from inefficiency due to redundant visual tokens. Existing pruning primary targets intra-frame spatial redundancy or prunes inside the LLM with shallow-layer overhead, yielding suboptimal spatiotemporal reduction and underutilizing long-context compressibility. All of them often discard subtle yet informative context from merged or pruned tokens. In this paper, we propose a new perspective that elaborates token \textbf{A}nchors within intra-frame and inter-frame to comprehensively aggregate the informative contexts via local-global \textbf{O}ptimal \textbf{T}ransport (\textbf{AOT}). Specifically, we first establish local- and global-aware token anchors within each frame under the attention guidance, which then optimal transport aggregates the informative contexts from pruned tokens, constructing intra-frame token anchors. Then, building on the temporal frame clips, the first frame within each clip will be considered as the keyframe anchors to ensemble similar information from consecutive frames through optimal transport, while keeping distinct tokens to represent temporal dynamics, leading to efficient token reduction in a training-free manner. Extensive evaluations show that our proposed AOT obtains competitive performances across various short- and long-video benchmarks on leading video LLMs, obtaining substantial computational efficiency while preserving temporal and visual fidelity. ### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：视频大语言模型（VLLM）在处理长视频时，视觉编码器生成的 Token 数量巨大（动辄数万甚至百万级），导致推理时的计算开销（FLOPs）和内存占用极高。
*   **研究动机**：
    *   **现有方法缺陷**：目前的 Token 缩减技术（如剪枝或合并）通常直接丢弃“不重要”的 Token，或仅在 LLM 内部进行浅层处理，这会导致细微但关键的语义信息丢失，且未能充分利用视频的时空冗余。
    *   **目标**：开发一种**无需训练（Training-free）**的方法，在大幅减少 Token 数量的同时，通过优化手段保留并聚合那些被剔除 Token 中的有效上下文信息。

### 2. 方法论：AOT（锚点最优传输）
核心思想是将 Token 缩减视为一个信息聚合问题，利用**最优传输（Optimal Transport, OT）**理论将冗余 Token 的特征“输送”到保留的锚点 Token 中。
*   **关键技术细节**：
    1.  **局部-全局锚点建立**：
        *   **全局锚点**：基于注意力机制（如 [CLS] Token 的注意力分数）选择最具代表性的 Token。
        *   **局部锚点**：采用网格划分（Grid-wise），在每个局部窗口内选择重要 Token，以保留细粒度的空间细节。
    2.  **阶段 I：帧内（Intra-Frame）优化**：
        *   将选中的锚点视为“需求方”，未选中的 Token 视为“供应方”。
        *   通过 **Sinkhorn-Knopp 迭代**快速求解最优传输矩阵，将未选中 Token 的语义信息按比例聚合到锚点中，实现空间压缩。
    3.  **阶段 II：帧间（Inter-Frame）优化**：
        *   将视频划分为片段（Clips），以片段首帧作为时间锚点。
        *   利用 OT 衡量后续帧与锚点帧的相似度，将相似信息聚合，仅保留变化剧烈（代表动态特征）的 Token，实现时间压缩。
*   **算法流程**：输入原始视频帧 -> 视觉编码 -> 建立锚点 -> 帧内 OT 聚合 -> 帧间 OT 聚合 -> 喂入 LLM 生成回答。

### 6. 主要结论与发现
*   **高效压缩**：在仅保留 **10%** 视觉 Token 的情况下，模型依然能保持原始性能的 **97.6%**。

### 7. 优点
*   **信息无损化倾向**：不同于简单的“丢弃”，它通过数学优化（OT）实现了信息的“浓缩”。
*   **即插即用**：无需微调模型权重，适配性强。
*   **时空兼顾**：同时解决了空间冗余和时间冗余，且锚点选择兼顾了全局视野和局部细节。

### 8. 不足与局限
*   **启发式边界**：帧间片段（Clip）的划分仍带有一定的启发式规则，对于镜头切换极快或极其复杂的视频，时间锚点的质量可能受限。
*   **极端压缩风险**：虽然在 10% 比例下表现良好，但在更极端的压缩下（如 <5%），OT 聚合可能会引入过多的混合噪声，导致语义模糊。
*   **应用限制**：目前主要在 LLaVA 系列模型上验证，对于其他架构（如基于 Mamba 或不同视觉编码器的模型）的泛化性有待进一步确认。
